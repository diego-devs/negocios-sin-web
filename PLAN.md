# 🌐 Proyecto: Webs para Negocios Locales

## Idea
Crear páginas web profesionales para negocios locales que no tienen presencia online, ofrecerlas ya hechas como demo, y cobrar ~$1,000 MXN (~$50 USD) por entregarlas.

## Zona objetivo
- **Base:** Colonia Nueva Santa María, Azcapotzalco, CDMX (CP 02800)
- **Expansión:** Colonias vecinas (Clavería, Pro-Hogar, Santo Tomás, etc.)

## Flujo de trabajo
1. **Prospección** → Identificar negocios sin web (directorios + recorrido presencial)
2. **Aprobación** → Diego aprueba con cuáles empezar
3. **Creación de demo** → Landing page personalizada (GitHub Pages, gratis)
4. **Contacto** → Correo/WhatsApp/visita presencial mostrando la demo
5. **Cierre** → Cobro y entrega del sitio
6. **Seguimiento** → Tracking en CSV

## Stack técnico
- HTML/CSS responsive (template base personalizable)
- Hosting: GitHub Pages (gratis)
- Sin dependencias de pago

## Pricing
- **$1,000 MXN** por sitio básico (landing page)
- Posibilidad de upsell: mantenimiento mensual, dominio propio, etc.

## Expectativas
- 3-5 demos por día
- 20+ contactos por semana
- Conversión esperada: 10-15% = 2-3 ventas/semana
- Ingreso potencial: $2,000-3,000 MXN/semana inicialmente

## Archivos
- `leads.csv` — Base de datos de todos los negocios (prospectos)
- `PLAN.md` — Este archivo
- `templates/` — Templates base de sitios web (por crear)
- `demos/` — Demos creadas para cada negocio (por crear)
